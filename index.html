📸 snappaste-pro - Paste images into your terminal instantly

Download snappaste-pro

snappaste-pro lets you paste screenshots directly into your Windows terminal. It works just like pasting text in a document. This tool connects your clipboard to tools like Claude Code, Gemini CLI, and GitHub Copilot. You save time by sending images to your AI assistant in one keystroke.

⚙️ How it works

Windows terminals do not support image pasting by default. This application detects when you press Ctrl+V after taking a screenshot. It captures the image data from your clipboard and formats it for terminal AI tools. You no longer need to save files to your desktop or drag images into folders.

📦 Installation and Setup

Follow these steps to set up the software on your computer:

  1. Visit this page to download the latest version of snappaste-pro.
  2. Locate the downloaded file in your Downloads folder.
  3. Double-click the file to start the installer.
  4. Follow the instructions on the screen to finish the setup.
  5. Launch the application from your Start menu.

🖥️ How to use the software

Once you run the application, it sits in your system tray. It monitors your clipboard automatically.

  1. Take a screen capture using your preferred method, such as the Windows Snipping Tool (Win+Shift+S).
  2. Open your terminal program.
  3. Press Ctrl+V inside the terminal window.

The application converts the image into text format that your AI assistant understands. It then pastes this data into your active command prompt.

✅ System Requirements

  • Windows 10 or Windows 11.
  • A supported terminal emulator like Windows Terminal, PowerShell, or Command Prompt.
  • An active AI coding assistant CLI installed on your machine.

🔧 Troubleshooting

If the application does not work, ensure it shows an icon in your system tray. Verify that your terminal has focus before you press Ctrl+V. If you restart your computer, the application launches automatically. You can close the application by right-clicking the icon in the system tray and choosing exit.
